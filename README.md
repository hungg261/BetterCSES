# BetterCSES

**BetterCSES** is a browser extension designed to enhance the user experience on the CSES Online Judge. This project is a fork of [CSES Helper](https://github.com/dada878/CSES-Helper).


## Key Features

The following enhancements are integrated directly into the CSES interface to streamline your competitive programming workflow.

| Feature | Description | Preview |
| :--- | :--- | :--- |
| **Statistics & TOC** | Dashboard navigation and progress tracking. | <img src="assets/extension/dashboard_toc.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Solved counter** | Problem count for each topic. | <img src="assets/extension/solved_counter.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Tags & Tips** | Supplemental problem categorizations and hints. | <img src="assets/extension/problem_tags_and_tips.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Problem sorting** | Sort the problem set by different criteria. | <img src="/assets/extension/sort_problems.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Flatten list** | Toggle a flat list view for all problems. | <img src="assets/extension/flatten_problem_list.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Show unsolved** | Filter for attempted tasks not yet solved. | <img src="assets/extension/show_unsolved.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Copy example** | One-click copy for sample inputs and outputs. | <img src="assets/extension/copy_example.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Quick submission** | Submit code via text area without file upload. | <img src="/assets/extension/submit_code.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Clipboard tools** | Copy source code and shared solutions. | <img src="/assets/extension/copy_code.png" width="400" style="border: 1px solid #ddd; border-radius: 4px; margin-bottom: 4px;"> <br> <img src="/assets/extension/copy_shared_code.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Translation** | Translate problem statements (Beta). | <img src="assets/extension/translate_languages.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Search problem** | Search problems by link, ID, or title. | <img src="assets/extension/search_problem.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
