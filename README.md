# BetterCSES

**BetterCSES** is a browser extension designed to enhance the user experience on the CSES Online Judge. This project is a fork of [CSES Helper](https://github.com/dada878/CSES-Helper).


## Key Features

The following enhancements are integrated directly into the CSES interface to streamline your competitive programming workflow.

| Feature | Description | Preview |
| :--- | :--- | :--- |
| **Statistics & TOC** | Track progress and navigate via a Table of Contents on the dashboard. | <img src="assets/extension/dashboard_toc.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Solved Counter** | View the number of solved problems for each specific topic. | <img src="assets/extension/solved_counter.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Tags & Tips** | Supplemental problem categorizations and hints to assist in solving. | <img src="assets/extension/problem_tags_and_tips.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Problem Sorting** | Sort the problem set based on different criteria for better navigation. | <img src="/assets/extension/sort_problems.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Flatten List** | Flatten the problem list for a more compact and direct view. | <img src="assets/extension/flatten_problem_list.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Show Unsolved** | Show attempted tasks you haven't solved yet. | <img src="assets/extension/show_unsolved.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Quick Submission** | Submit code via text area directly without the need to upload files. | <img src="/assets/extension/submit_code.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Clipboard Tools** | One-click copy for both your own code and shared solutions. | <img src="/assets/extension/copy_code.png" width="400" style="border: 1px solid #ddd; border-radius: 4px; margin-bottom: 4px;"> <br> <img src="/assets/extension/copy_shared_code.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
| **Translation** | Beta support for translating problem statements into other languages. | <img src="assets/extension/translate_languages.png" width="400" style="border: 1px solid #ddd; border-radius: 4px;"> |
